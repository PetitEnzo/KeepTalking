# Images de profil

Ajoutez vos images de profil dans ce dossier.

## Format recommandé
- Format : PNG ou JPG
- Taille : 200x200px minimum
- Noms de fichiers : avatar1.png, avatar2.png, avatar3.png, etc.

## Images actuellement disponibles
Pour le moment, le système utilise des emojis par défaut.
Ajoutez vos propres images ici pour les utiliser comme avatars.

## Exemple de structure
```
profile_pictures/
  ├── avatar1.png
  ├── avatar2.png
  ├── avatar3.png
  └── ...
```
