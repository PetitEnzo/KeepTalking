# Assets

Ce dossier contient les ressources statiques de l'application.

## Fichiers requis

### Icônes et images

- **icon.png** (1024x1024) - Icône de l'application
- **splash.png** (1242x2436) - Écran de démarrage
- **adaptive-icon.png** (1024x1024) - Icône adaptative Android
- **favicon.png** (48x48) - Favicon pour le web

## Générer les assets

Vous pouvez créer vos propres assets ou utiliser des outils en ligne :

1. **Icône de l'application** : Créez une image 1024x1024 avec votre logo
2. **Splash screen** : Créez une image 1242x2436 pour l'écran de chargement
3. **Favicon** : Réduisez votre icône à 48x48 pour le web

### Outils recommandés

- [Figma](https://figma.com) - Design d'icônes
- [Canva](https://canva.com) - Création rapide
- [Icon Kitchen](https://icon.kitchen/) - Générateur d'icônes

## Placeholder temporaires

Pour le développement, vous pouvez utiliser des images placeholder :
- Créez des carrés de couleur unie
- Utilisez des générateurs d'images comme [placeholder.com](https://placeholder.com)

## Note

Les fichiers PNG sont actuellement des placeholders. Remplacez-les par vos vrais assets avant le déploiement en production.
